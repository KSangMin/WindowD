키보드로 이동 및 점프, 상호작용

마우스로 회전 및 상호작용

체력, 스태미너, 속도 등의 UI

속도, 이단 점프, 무적 등의 아이템

아이템 조사 UI

아이템 장착 UI

점프대, 이동 플랫폼, 레이저 트랩

벽타기

이동 파티클

오브젝트와의 상호작용
